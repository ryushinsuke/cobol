   cobol等程序运行过程中，发生的错误信息以Email的形式发送到客户邮箱。
2，DB最优化
   对数据库中表以及分区表的容量进行最优化处理
3，cobol性能改善
   通过在section前后添加时间点的方法来判断某section花费的时间比较多，然后对该节进行优化。
   主要从优化SQL，减少调用子程序次数及减少I/O考虑。
